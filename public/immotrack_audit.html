⚠️ Points d'attention

Suppression compte — Admin supprime le profil mais auth.users reste. Action manuelle requise : Supabase > Authentication > Users > Delete.
Quittances Storage — Nécessite un bucket "documents" dans Supabase Storage (public read).
Email invitation — Fonctionne avec le SMTP par défaut Supabase. En production, configurer un SMTP personnalisé.
SMTP Supabase — Le rate-limit par défaut est 3 emails/heure. Configurer un service SMTP en production (Sendgrid, Resend, etc.).
